POST DocumentFrontSide

Request

Field Type Required Description Values
SubCustomer string Yes Defines the name of the subclient configured on the platform Text string
Uid string No This field receive the uid generated in before steps Text string
Image string Yes Receives the base64 image of the front of the document Text string en base64
InvitationKey string Yes Unique identifier of the invitation Text string
Email string No Email associated to the user who creates the invitation Text string
numberPhone string No Number phone of the user created in the invitation. Text string
DeviceId integer($int32) No Identifier registered on the device platform Returns an Integer
documentType integer($int32) No User document type Returns an Integer
maxDocumentAttempts boolean No If a maximum number of attempts was set for document upload. True or false
Authorization string Yes This field receive a bearer token for authentication,bearer token Text string
