POST GetToken

Request

Field Type Required Description Values
grant_type string Yes Type of autorization Insert word: "Password"
username string No Receives the configured username or Sub Customer Text string
password string No Password associated with the user or sub customer in SHA1 format Text string
Authorization string Yes This field receive a bearer token for authentication,bearer token Text string

Example

  • curl -x \ 'https://sandbox.idfactory.me/Authorization/api/GetToken' \
  • -H 'accept: application/json' \
  • -H 'Content-Type: application/x-www-form-urlencoded' \
  • -d 'grant_type=password&username=USERNAME&password=SHA1PASSWORD'

Note:

  • In the "USERNAME field replace with your username.
  • In the "SHA1PASSWORD" field replace with the password in SHA1 format.

200 OK
