GET GetInvitationByKey

Retrieves invitation details by providing a specific invitation key.

Request

Field Type Location Required Description
SubCustomer string Header Yes Sub customer identifier
key string Query Parameter Yes Unique invitation key identifier
Authorization string Header Yes Bearer token for authentication

Example

  • curl -X 'GET' \
  • 'https://sandbox.idfactory.me/Invitations/api/GetInvitationByKey?key=invitationKey' \
  • -H 'accept: application/json' \
  • -H 'SubCustomer: SubCustomerName' \
  • -H 'Authorization: bearer TOKEN'

Note:

  • Replace "SubCustomerName" with your actual SubCustomer Name.
  • Replace "TOKEN" with your actual bearer token.
  • Replace the key parameter with the actual invitation key.
