POST Verify

Request

Field Type Required Description Values
SubCustomer string Yes In this field, the name of the client was received with which the person's identity will be verified. Text string
PersonId string No Identifier of the identity, generated for the system. Text string
IdNumber string No Identity identification number Text string
caseId string No This field receive the case id generated in liveness process Text string
invitationKey string Yes This field receive the invitation previously created in Invitations/Created endpoint. Text string
deviceMetadata string No Device metadata Text string
csid string No Customer session ID Text string
deviceId integer($int32) No Identifier registered on the device platform Returns an integer
riskId integer($int32) No Risk level defined for the transaction Returns an integer
closeAttempts integer($int32) No Number of attempts configured by subcustomer to take the photo. Integer
Authorization string Yes This field receive a bearer token for authentication,bearer token Text string
