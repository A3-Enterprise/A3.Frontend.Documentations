POST Close

Request

Field Type Required Description Values
subCustomer string Yes This field received the customer’s name with wich it will be carried out the invitation for will be registered on the system. Text string
uid string No This field receive the uid generated in before steps. Text string
authorization string Yes This field receive a bearer token for authentication,bearer token Text string

Example

  • curl -X \
  • 'https://sandbox.idfactory.me/Enroll/api/Close' \
  • -H 'accept: application/json' \
  • -H 'subCustomer: CajaMetropolitanaSubCustomerQA' \
  • -H 'uid: FC3C8FB492DF4BFB82E59448051FDF7812072024014614640' \
  • -H 'authorization: bearer gIAE8XbkJAgTakwCvFFA-50qXeDJLa9rqP1aQydozHSLf0fQXd82Xwbnxq_MqIsggs0t2S2QhDzXFil23C2fv8-glZM4KtS-AYha1zSJG9sZGk9Gvlm82Cayp7I8Pq3D7C1xU0pa0ywvBs35AadeqaWATyu_eVF-TR8yangNNm_chcm6LKloRE8YTjdSD_cletxE80gtIxhiEAlAeyonoGJXXnGuD2kTlTQStSNWPWnUvEHAjwPin2fZybKKWM6eRZRPoD6ftXiXhre25-8eZZ8eumkulA0UJlGrKvwRPL9vdPWujo4junPo2W3tD_L0ApKvDybBarBMZz7SKZypZA'
